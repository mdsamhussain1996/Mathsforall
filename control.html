State-Space Models

A unified mathematical framework for dynamical systems

The State-Space Representation

Any finite-dimensional linear dynamical system can be written in state-space form. The state x(t) contains all information needed to predict future behavior. The matrices A, B, C, D fully characterize the system.

ẋ(t) = Ax(t) + Bu(t) [State equation] y(t) = Cx(t) + Du(t) [Output equation] x ∈ ℝⁿ: state vector u ∈ ℝᵐ: input, y ∈ ℝᵖ: output

Transfer Functions vs State Space

Transfer functions G(s) = C(sI−A)⁻¹B + D describe SISO systems in the frequency domain. State-space handles MIMO systems, nonlinear extensions, and gives direct access to internal states — essential for modern control design.

Solution of Linear Systems

The solution is x(t) = e^{At}x₀ + ∫₀ᵗ e^{A(t-τ)}Bu(τ)dτ. The matrix exponential e^{At} — the "system transition matrix" — encodes how states evolve from initial conditions. Its eigenvalues determine stability.

e^{At} = I + At + (At)²/2! + (At)³/3! + … Eigenvalues of A → poles of the system

Stability Analysis

Lyapunov methods, eigenvalue analysis, and stability regions

Lyapunov Stability

Lyapunov's direct method proves stability without solving the ODE. Find a Lyapunov function V(x) — a "generalized energy" that is positive definite and whose time-derivative is negative definite along system trajectories.

V(x) > 0 ∀x ≠ 0 (positive definite) V̇(x) ≤ 0 ∀x (non-increasing) If V̇(x) < 0: asymptotically stable If V̇(x) ≤ 0: stable (Lyapunov sense)

Linear Systems: Eigenvalue Criterion

For linear systems ẋ = Ax, stability is determined entirely by eigenvalues of A. All eigenvalues must have negative real parts for asymptotic stability. This connects directly to pole placement in control design.

Stable: Re(λᵢ(A)) < 0 ∀i Unstable: ∃i: Re(λᵢ(A)) > 0 Marginal: max Re(λᵢ(A)) = 0

Controllability & Observability

A system is controllable if we can drive it from any state to any other. It is observable if we can determine the full state from outputs alone. These concepts are dual to each other.

𝒞 = [B AB A²B … Aⁿ⁻¹B] rank(𝒞) = n 𝒪 = [C; CA; CA²; …; CAⁿ⁻¹] rank(𝒪) = n

Phase Portraits

Geometric view of dynamical system trajectories

What is a Phase Portrait?

A phase portrait shows the trajectories of a dynamical system in the state space (x₁, x₂). Each point shows the velocity vector (ẋ₁, ẋ₂) — the direction the system will evolve from that state. Equilibria appear as fixed points.

Types of Equilibria

The character of an equilibrium is determined by the eigenvalues of the Jacobian matrix at that point:

  • Stable node: real negative eigenvalues
  • Stable spiral: complex with negative real part
  • Center: purely imaginary eigenvalues
  • Unstable node/spiral: positive real parts
  • Saddle: eigenvalues of opposite signs
